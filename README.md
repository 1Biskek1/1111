Dashcoin Wallet GUI
=================

Wallet GUI for Dashcoin.

- Syncs blockchain with network
- Can load/generate a wallet
- Loads balance
- Loads address
- Shows transaction history
- Sends Dashcoin


Windows builds: https://sourceforge.net/p/dashcoinwalletgui/
